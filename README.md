🌳 LBLAD - Structure App V1
LBLAD est un assistant IA intelligent conçu pour le fellah marocain, permettant de diagnostiquer les maladies de l'olivier par la voix et l'image.

🚀 Fonctionnalités Clés :
IA Multimodale : Analyse instantanée d'une photo et d'une description vocale en Darija.

Practical AI : Fournit un diagnostic précis et une "ordonnance" concrète (traitements et prévention).

Data-Driven : Basé sur un dataset propriétaire extrait des rapports officiels de l'INRA et l'ONSSA.

🛠️ Stack Technique :
Interface : Figma (Design Mobile intuitif).

Data : Mapping JSON (Jargon local ↔ Expertise scientifique).

Logic : Architecture RAG pour des réponses fiables et localisées.
